# Knowledge base documents

The domain documents used to build ENERA's knowledge base are **official public publications** and are **not redistributed** in this repository. They remain the property of their respective issuers.

To reproduce the knowledge base, download the five documents from their official sources below and place the PDFs in this `data/` folder, then run `indexer.py`. Only the content-relevant page ranges defined in `metapodaci_dokumenti.xlsx` are indexed.

> The MIT license of this repository applies to the **source code only**, not to these documents.
